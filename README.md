# Uri Guide Website

Uri Guide is a prototype website for a multilingual tourism and cultural guide platform in South Korea.

The project focuses on helping international students, foreign workers, and CIS/Russian-speaking travelers experience Korea with local explanations, language support, and small-group cultural tours.

## Main Idea

Uri Guide connects visitors with local guides who understand their language, culture, and practical needs in Korea.

## Current Features

- Landing page structure
- Hero section
- Basic navigation
- Separate CSS and JavaScript files
- Responsive design preparation

## Planned Features

- Tour package section
- Student and worker support section
- Booking request form
- Guide matching logic
- Multilingual content
- Customer reviews and testimonials
